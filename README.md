# Rechi

Aplicação construída com Laravel 13 e o starter kit oficial React.

## Stack

- Laravel 13 e PHP 8.3
- React 19 e TypeScript
- Inertia 3
- Tailwind CSS 4
- shadcn/ui
- Laravel Fortify

## Autenticação

- Login e cadastro
- Recuperação de senha
- Verificação de e-mail
- Autenticação em dois fatores
- Passkeys

## Desenvolvimento

```bash
composer run dev
```

## Verificações

```bash
composer test
npm run build
```
